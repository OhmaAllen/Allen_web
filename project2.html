Book Recommendation System

📅 Duration: 10.2024 - 12.2024

📍 AI & Machine Learning Project

Project Overview

This project is an AI-powered book recommendation system that utilizes a vector-based recommendation algorithm to help users discover new books based on their reading preferences and review history.

Features

  • 📖 **Vector-based book recommendation algorithm**
  • 👤 **User profile & review management**
  • 📚 **Comprehensive book and author tracking**
  • 💻 **Command-line interface for easy interaction**
